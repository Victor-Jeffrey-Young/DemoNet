# 阶段 0-5：项目骨架与基础功能

> **状态**：✅ 已完成

---

## 阶段 0：项目骨架

- 初始化 Spring Boot + Vue 3 项目
- Docker Compose 编排 MySQL + Redis + RabbitMQ
- 前端 Vite + Tailwind CSS 4 + Element Plus

## 阶段 1：CRUD API

- ItemController 实现 RESTful CRUD
- MyBatis-Plus 分页查询
- 自定义 slug 生成逻辑

## 阶段 2：用户系统

- Spring Security + JWT 无状态认证
- 注册/登录/获取当前用户
- 密码 BCrypt 加密存储
- 默认管理员 `admin / changeme`

## 阶段 3：搜索与推荐

- 标签系统（tags + item_tag_mapping 多对多）
- 按品类/标签/关键词搜索
- 热门推荐列表排序算法

## 阶段 4：异步抓取管道

- RabbitMQ 消息队列：FetchConsumer 消费抓取任务
- SteamService：调用 Steam Store API 拉取应用详情
- TMDBService：调用 TMDB API v3 拉取电影信息
- Python 脚本批量导入种子数据

## 阶段 5：8 品类扩展

- 从 4 品类扩展到 8 品类（game/movie/anime/boardgame/model/book/music/digital）
- 每种品类有独立类型标识和元数据
- 后续追加 2 品类（coffee/experience）达到 10 品类
