## 二、视觉设计：CoverFlow 封面墙

### CoverFlow 概念

经典 iTunes CoverFlow：专辑封面以 3D 透视排列，中央最大、两侧逐渐旋转缩小，形成深度空间感。

```
         ╱              ╲
       ╱     ┌────┐       ╲
     ╱       │封面│         ╲
   ╱  ┌────┐ │中央│ ┌────┐   ╲
 ╱    │封面│ │    │ │封面│     ╲
│     │左侧│ └────┘ │右侧│      │
│     └────┘        └────┘      │
│                               │
│   ◄ prev        next ►       │
│     ○  ○  ●  ○  ○            │
└───────────────────────────────┘
```

### CSS 实现方案

```css
perspective: 900px on container
cards:
  [prev-2] rotateY(45deg) translateZ(-200px) scale(0.7)   opacity: 0.3
  [prev-1] rotateY(30deg) translateZ(-100px) scale(0.85)  opacity: 0.6
  [active] rotateY(0)    translateZ(0)     scale(1)       opacity: 1
  [next+1] rotateY(-30deg) translateZ(-100px) scale(0.85) opacity: 0.6
  [next+2] rotateY(-45deg) translateZ(-200px) scale(0.7)  opacity: 0.3
```

- 5 张可见封面，左右各 2 张渐远
- 点击两侧封面平滑切换到中央（`transition: all 0.5s cubic-bezier`）
- 使用 `transform-style: preserve-3d` 保持 3D 空间
- 反射倒影：每张封面下方 `scaleY(-1)` + 渐变遮罩（可选）

## 三、MusicHero — CoverFlow 封面墙

替换全屏轮播，音乐频道顶部使用 CoverFlow：

| 属性 | 说明 |
|------|------|
| 数据源 | `getFeatured({type:'music'})` → fallback `getHotItems()` |
| 背景 | fuchsia/pink 暗色渐变 + 粒子/光晕装饰 |
| 封面数 | 5 张可见 + 无限循环（首尾相接） |
| 交互 | 点击两侧封面 → 切换为中央；dot 指示器；左右箭头 |
| 自动播放 | 5s 自动轮播 |
| 目录栏 | 下方显示当前专辑名 + 艺人 |

### Detail.vue 音乐专用布局（已实现）

```
┌─────────────────────────────────────────────┐
│  ┌───────────────────────────────────────┐  │
│  │                                       │  │
│  │     Apple Music 嵌入 iframe            │  │  ← 优先：externalLink → embed.music.apple.com
│  │     （或 专辑封面 + 音频播放器）          │  │  ← fallback：preview_url + coverUrl
│  │                                       │  │
│  └───────────────────────────────────────┘  │
│                                             │
│  🎵 音乐    itunes                          │
│  专辑名                           [收藏 ▼]  │
│  艺人 · N tracks                           │
│                                             │
│  ┌─────────┬─────────┬─────────┬─────────┐  │
│  │  艺人   │  年份   │  风格   │  曲目   │  │
│  └─────────┴─────────┴─────────┴─────────┘  │
│                                             │
│  [🎵 在 Apple Music 中打开]                  │  ← externalLink 跳转按钮
│                                             │
│  🎵 试听预览  ──▶ ═══════════              │  ← audio 控件（preview_url）
└─────────────────────────────────────────────┘
```

**播放器优先级**：Bilibili/YouTube 视频 > Apple Music 嵌入 > 专辑封面+音频播放器 > emoji 占位

**Apple Music 嵌入**：自动将 `externalLink` 中的 `music.apple.com` 替换为 `embed.music.apple.com`，生成 iframe 嵌入 URL，类似 YouTube 视频嵌入体验。容器高度固定 `h-[450px]` 匹配 widget 实际尺寸。

**专辑封面播放器**：当没有 Apple Music 链接但有 `preview_url` 时，hero 区展示模糊化专辑封面背景 + 清晰封面图 + 歌名/艺人 + `<audio>` 控件。容器使用 `py-12` 自适应内容高度。

**YT/B站 按钮**：音乐详情页隐藏视频源切换按钮（`v-if="!isMusic"`），避免与音乐播放器风格冲突。
