## TMDB 数据管道

### API 调用

```
GET /3/movie/{tmdbId}?append_to_response=videos,credits&language=zh-CN
```

- **Video 获取注意事项**：`language=zh-CN` 参数会导致 YouTube 预告片被屏蔽！
- 解决：先用中文参数获取详情（简介/导演等），再用**不带 language 参数**的请求获取 `videos`
- 筛选条件：`site=YouTube && type IN (Trailer, Teaser)`

### 提取字段

| 字段 | API 来源 |
|------|---------|
| 海报 `w500` | `movie.poster_path` |
| 背景 `w1280` | `movie.backdrop_path` |
| 中文简介 | `movie.overview` |
| 导演 | `credits.crew[ job=Director ]` |
| 年份 | `movie.release_date` 截取前 4 位 |
| 时长 | `movie.runtime` + "min" 后缀 |
| 类型 | `movie.genres[].name` 逗号拼接 |
| 预告片 | `videos.results` 筛选 YouTube + Trailer/Teaser |

### 批量导入

Python 脚本通过已知 TMDB 电影 ID 批量拉取：
- 自动去重（按 tmdb_id 唯一）
- 清理噪声数据
- 自动填充 `info_json`、`cover_url`、`wide_cover_url`、`external_link`
