## 本地开发

### 环境准备

```bash
# 首次：生成 .env 文件
bash scripts/setup-env.sh

# 启动基础设施（MySQL + Redis + RabbitMQ）
docker compose up -d

# 后端 (:8080)
cd backend && mvn clean spring-boot:run

# 前端 (:5173)
cd frontend && npm run dev
```

| 服务 | 地址 |
|------|------|
| 前端 | http://localhost:5173 |
| 后端 API | http://localhost:8080 |
| Swagger UI | http://localhost:8080/swagger-ui/index.html |
| RabbitMQ 管理界面 | http://localhost:15672（admin/changeme） |
| 默认管理员 | admin / 见 `ADMIN_DEFAULT_PASSWORD` 环境变量 |

### docker-compose.yml（开发）

| 服务 | 镜像 | 端口 |
|------|------|------|
| MySQL | `mysql:8.0` | 3306 |
| Redis | `redis:7-alpine` | 6379 |
| RabbitMQ | `rabbitmq:3.12-management` | 5672 / 15672 |

> 启动时自动挂载 `schema.sql` 建表（`CREATE TABLE IF NOT EXISTS`，幂等安全）。

## 环境变量 (.env)

复制 `.env.example` 为 `.env` 并填入真实值。

### 基础设施

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `MYSQL_HOST` | localhost | 数据库主机 |
| `MYSQL_USER` | root | 数据库用户 |
| `MYSQL_PASSWORD` | changeme | 数据库密码（**生产必须修改**） |
| `MYSQL_ROOT_PASSWORD` | changeme | MySQL root 密码（**生产必须修改**） |
| `REDIS_HOST` | localhost | Redis 主机 |
| `RABBITMQ_HOST` | localhost | RabbitMQ 主机 |
| `RABBITMQ_USER` | admin | RabbitMQ 用户 |
| `RABBITMQ_PASS` | changeme | RabbitMQ 密码（**生产必须修改**） |

### 应用安全

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `JWT_SECRET` | （内置默认值） | JWT 签名密钥（**生产必须修改为 32 字节以上随机字符串**） |
| `ADMIN_DEFAULT_PASSWORD` | changeme | 管理员初始密码（**生产必须修改**） |
| `UPLOAD_DIR` | ./uploads | 文件上传目录 |

### 外部 API 密钥

| 变量 | 说明 | 获取方式 |
|------|------|----------|
| `STEAM_API_KEY` | Steam Web API Key | https://steamcommunity.com/dev/apikey |
| `TMDB_API_KEY` | TMDB API v3 | https://www.themoviedb.org/settings/api |
| `IGDB_CLIENT_ID` | Twitch IGDB Client ID | https://dev.twitch.tv/console |
| `IGDB_CLIENT_SECRET` | Twitch IGDB Client Secret | 同上 |
| `STEAMGRIDDB_API_KEY` | SteamGridDB（竖版封面） | https://www.steamgriddb.com/profile/preferences/api |

### 可选功能

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `TURNSTILE_SITE_KEY` | 空 | Cloudflare Turnstile 前端 key（空=跳过人机验证） |
| `TURNSTILE_SECRET_KEY` | 空 | Cloudflare Turnstile 后端验证 key |
| `INVITE_ONLY` | false | 是否开启邀请码注册（通过管理后台动态调整） |

> **注**：API 密钥也可通过管理后台「API 设置」标签页动态更新（存储在 `app_settings` 表），无需重启服务器。环境变量在服务启动时作为初始值，数据库中的值优先级更高。

## 生产部署

### docker-compose.prod.yml（5 服务）

| 服务 | 说明 |
|------|------|
| `db` | MariaDB 11（替代 MySQL 8.0，生产更稳定） |
| `redis` | Redis 7-alpine |
| `rabbitmq` | RabbitMQ 3-management-alpine |
| `backend` | Spring Boot JAR → `backend/Dockerfile` |
| `frontend` | Nginx 静态文件 → `frontend/Dockerfile` + `nginx.conf` |

### 构建与启动

```bash
# 首次：生成 .env（填入生产密码和 API 密钥）
bash scripts/setup-env.sh

# 构建 + 启动全部服务
docker compose -f docker-compose.prod.yml up -d --build

# 查看日志
docker compose -f docker-compose.prod.yml logs -f backend
```

### Nginx 配置要点

- 前端静态文件（`dist/`）由 Nginx 直接 serve
- `/api/` 路径反向代理到 `backend:8080`
- `/uploads/` 路径由后端 serve（或 Nginx 直接映射上传目录）
- 建议配置：gzip 压缩 + 静态资源缓存头 + `try_files $uri $uri/ /index.html`（SPA 路由支持）

## 回滚方案

```bash
# 停止当前服务
docker compose -f docker-compose.prod.yml down

# 恢复数据库（如有备份）
mysql -u root -p demonet < backup_YYYYMMDD.sql

# 启动上一版本镜像（git checkout + rebuild）
git checkout <上一个稳定 tag>
docker compose -f docker-compose.prod.yml up -d --build
```
