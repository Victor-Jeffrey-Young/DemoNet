# 问题日志

> 按阶段分组，每个问题包含：现象 → 根因 → 解决方案

---

## 基础设施（阶段 0-5）

### 1. mvnw 缺失
- **现象**：项目中无 `mvnw` 脚本
- **解决**：直接用全局 `mvn` 命令

### 2. MyBatis-Plus 依赖版本
- **现象**：启动报错，找不到 `MybatisPlusInterceptor`
- **解决**：使用 `mybatis-plus-spring-boot4-starter:3.5.16`

### 3-4. 分页 total=0
- **现象**：分页返回 total=0，但数据正常
- **根因**：缺少 SQL 解析依赖
- **解决**：添加 `mybatis-plus-jsqlparser:3.5.16`

### 5. JDBC 中文编码
- **解决**：`characterEncoding=UTF-8` 配置

### 6. 中文双重编码
- **现象**：Python 脚本插入中文后出现双重编码
- **解决**：Python 使用 `mysql-connector-python` 直连，设置 charset

### 7. VSCode ClassNotFoundException
- **现象**：IDE 运行报类未找到
- **解决**：`mvn clean` 清除缓存后重试

### 8. docker-compose version
- **现象**：新版 Docker Compose 报 `version` 已弃用
- **解决**：移除 `docker-compose.yml` 中的 `version` 行

### 9. Map.of 参数限制
- **现象**：超过 10 个条目编译报错
- **解决**：改用 `Map.ofEntries()`

### 10. 收藏下拉菜单消失
- **现象**：CSS `group-hover` 在快速移动鼠标时丢失状态
- **解决**：JS `mouseenter/mouseleave` 事件替代

### 11. Profile 页面无标题
- **现象**：收藏列表只显示 ID，不显示标题
- **解决**：`UserItemService` 使用 JOIN 查询 items 表返回 title/slug

### 12. seedTags() 空异常
- **现象**：查询空表报 NPE
- **解决**：`queryForList` 替代 `queryForObject`

### 13. 详情页 overflow-hidden 裁剪下拉
- **现象**：收藏下拉菜单被卡片 overflow-hidden 裁剪
- **解决**：下拉菜单移至图片区域外部

## 游戏频道（阶段 7.1）

### 14. 轮播图卡顿
- **现象**：`scrollTo` 滑动不流畅，明显掉帧
- **根因**：`scrollTo` 触发浏览器重排
- **解决**：改用 `transform: translateX()` GPU 加速

### 15. 轮播克隆闪白
- **现象**：边缘跳转时出现白屏闪烁
- **根因**：2 份克隆不够，跳转时 DOM 切换可见
- **解决**：3 份全量拷贝（24 张 DOM），中间区操作，边缘跳转不可见

### 16. Steam 图裁切
- **现象**：Steam 头图被裁剪过多
- **解决**：`aspect-[16/10]` → `aspect-[21/10]` 匹配真实比例；`bg-center` → `bg-top` 保留标题区域

### 17. @Value 缺失 import
- **现象**：SteamService 编译报错
- **解决**：添加 `import org.springframework.beans.factory.annotation.Value`

### 18. 竖版卡片缺图
- **现象**：GameCard 竖版布局需要竖版图，但 Steam 只有横版头图
- **当前方案**：暂用横版图 `bg-top` 对齐顶部
- **长期方案**：见阶段 8.6（管理员上传竖版海报）

## 用户/导航

### 19. 默认用户丢失
- **现象**：数据库重建后 admin 用户不存在
- **根因**：`seedUser()` 被 `items` 判断条件包裹，`items` 为空时跳过了用户创建
- **解决**：`seedUser()` 移到 `items` 判断之前独立执行

### 20. Navbar 用户状态消失
- **现象**：登录后刷新页面，Navbar 恢复未登录状态
- **根因**：Vue 组件挂载时未恢复用户状态
- **解决**：App.vue `onMounted` 调用 `auth.fetchUser()` 从 localStorage 恢复

### 21. List 头部描述重复
- **现象**：`meta.desc` 与标题显示内容雷同
- **解决**：改为格式化文本 `共 X 条内容`

## 视频 & 数据

### 22. 视频过早播放
- **现象**：进入详情页后 iframe 自动播放，抢占视觉注意力
- **解决**：iframe 移到卡片头部 emoji 占位区域，与信息卡片整合

### 23. async function setStatus 被误删
- **现象**：Admin.vue 编译报错
- **根因**：编辑器无意中删除了函数声明行
- **解决**：恢复函数声明

### 24. TMDB 网络不通
- **现象**：终端 `curl api.themoviedb.org` 超时
- **根因**：中国大陆网络环境不稳定
- **当前状态**：偶尔可恢复，需要重试机制

### 25. language=zh-CN 使 videos 为空
- **现象**：带 `language=zh-CN` 请求 TMDB 时 `videos` 数组为空
- **根因**：TMDB 中文语言参数会过滤掉 YouTube 英文预告片
- **解决**：必须**不带 language 参数**单独请求 `append_to_response=videos`，再用 `site=YouTube && type IN (Trailer,Teaser)` 筛选

### 26. TMDB 搜索噪声
- **现象**：`search/movie?query=Oppenheimer` 返回 20+ 纪录片/变体
- **解决**：先搜索获取精确 ID，再用 `fetchMovieDetail(tmdbId)` 获取单一结果；导入后用 `GROUP BY title HAVING COUNT>1` 清理重复

### 27. 电影外键约束
- **现象**：删除电影时违反外键约束
- **根因**：`user_items` 表关联 `items.id`
- **解决**：先删 `user_items` 中关联记录，再删 `items`

### 28. AppCard 封面图遗失
- **现象**：所有卡片只显示 emoji，不显示封面图
- **根因**：重写类型映射时漏掉了 `v-if="item.coverUrl"` 条件渲染
- **解决**：恢复封面图 `bg-cover` + emoji 兜底显示
