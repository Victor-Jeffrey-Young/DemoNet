# 数据模型

## 数据库表

6 表：`items` / `users` / `user_items` / `tags` / `item_tag_mapping` / `reviews`

### items 表字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | BIGINT PK | 自增主键 |
| `type` | VARCHAR | 品类：game/movie/anime/boardgame/... |
| `title` | VARCHAR | 标题 |
| `slug` | VARCHAR UNIQUE | URL 友好标识 |
| `cover_url` | VARCHAR | 横版封面图（Steam 头图/TMDB 背景） |
| `wide_cover_url` | VARCHAR | 宽幅封面（备选） |
| `poster_url` | VARCHAR | 竖版海报图（管理员上传） |
| `description` | TEXT | 描述 |
| `info_json` | JSON | 品类特定结构化数据 |
| `media_url` | VARCHAR | 视频/媒体 URL |
| `external_link` | VARCHAR | 外部商店链接 |
| `external_id` | VARCHAR | Steam AppID / TMDB ID |
| `source` | VARCHAR | 数据来源：steam/tmdb/manual |
| `status` | VARCHAR | published/pending/rejected |
| `created_at` / `updated_at` | DATETIME | 时间戳 |
