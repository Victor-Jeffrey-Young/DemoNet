## 品类设计详情

### 1. 游戏频道

**Hero：无限循环轮播**
- 3 份全量拷贝（24 张 DOM），中间区操作，边缘跳转不可见
- `transform: translateX()` GPU 加速，`requestAnimationFrame` 控制过渡
- 3.5s 自动播放，dot 指示器可点击
- 封面比例 `aspect-[21/10]` 匹配 Steam 头图

**Card：竖版卡片**
- 380px 宽度，hover 内容滑入视差（700ms cubic-bezier）
- DEMO 徽章：彩虹渐变边框脉冲动画，仅 `demo_available=true` 显示
- 双视频源嵌入：优先 Bilibili → YouTube，右上角切换按钮

---

### 2. 电影频道

**Hero：全屏 Banner**
- `h-[calc(100vh-4rem)]` 全屏高度
- 900ms 渐入过渡，5s 间隔自动播放
- 深红左侧渐变遮罩 + 标题/年份/类型/简介

**Card：横向宽卡 + 3D 翻转海报卡**
- 横卡：1/3 TMDB 海报 + 2/3 信息区，hover 海报放大 110%
- 翻转卡：`perspective` + `rotateY(-78°)`，hover 旋转露出背面信息（导演/评分）

**TMDB 管道**
- `GET /3/movie/{tmdbId}?append_to_response=videos,credits`
- `language=zh-CN` 会屏蔽 YouTube 预告片 → 中文参数获取详情，不带 language 获取 videos

---

### 3. 动漫频道

**Hero**：紫/粉调轮播，AniList + Bangumi + TMDB TV 三数据源
**Card**：灰白底卡片，hover 时 `filter: grayscale(100%)`，标题和标签同步上移

### 4. 桌游频道

**Hero：游戏店橱窗**
- 暖琥珀渐变遮罩（`from-amber-950 via-yellow-950/70 to-gray-950`）
- 木桌 + 毛毡垫质感

**Card：盒装展示卡**
- 横版宽卡：左侧 1/3 盒图 + 右侧信息区
- 人数 👥 + 时长 ⏱ + 重量 ⭐（★ 可视化，1.0→1★, 3.9→4★）
- 机制标签 chips + 「▶ 规则教学」视频入口
- "规则书"折叠面板：翻页式图片 + 文字规则 + 灯箱全屏

### 5. 模型频道

**Hero：格纳库陈列架** — 暗色背景 + 荧光绿色调，陈列架层板
**Card：全宽画廊卡** — 等级/比例/材质/系列标注

### 6. 书籍频道

**Hero：书店展示台**
- 首个浅色品类（`light:true`），页面暖奶油 `#faf6f1`
- Hero 深木色渐变 + `radial-gradient` 柔光聚光
- 3-4 本书封面朝上平放（`aspect-[3/4]` + 3D 书脊）
- 歪斜手写推荐卡（`transform: rotate(-1deg)`, `font-serif italic`）

**Card：书架展示卡**
- 3D 书脊 + 顶部高光线，hover `translateX(4px)` 模拟抽书
- 书架隔板纯 CSS 装饰线
- 作者 · 页码 · 年份暖棕色徽章

### 7. 音乐频道

**Hero：CoverFlow 3D 封面墙**

- 拖拽 + 自动轮播 + 倒影效果
- v1→v5 五次重构，逐步完善交互

**Card：正方形专辑卡**
- iTunes 真实封面 + Apple Music 嵌入 iframe
- 详情页：Apple Music 播放器 + "在 Apple Music 中打开"按钮

### 8. 数码频道

### 9. 咖啡频道

**Hero：黑板手写菜单**
- 黑板背景（`bg-gray-900`）+ 粉笔手写字体
- 菜单布局：品名/产区/风味，白色粉笔字 + 装饰性粉笔涂鸦
- 咖啡豆信息面板（产地/烘焙度/处理法/品种）
- 风味标签（柑橘/茉莉/蜂蜜等，彩色 chips）
- 品鉴视频 iframe 嵌入（B站/YT）

**Card：咖啡豆标签卡**
- 牛皮纸质感底 + 木质相框或软木板 pin
- hover 微旋转 0.5°，模拟纸质标签的自然姿态
- "品鉴视频 →" 入口

**「试玩」体验**：冲煮指南 `brew_guide`（水温/粉水比/时间/步骤）📋 待实现

### 10. 线下体验频道

**Hero：软木公告板**
- 软木纹理底（`bg-[#c4a47a]` + `repeating-linear-gradient` 模拟颗粒）
- 红色图钉 📌 四角装饰 + 麻绳 + 小木夹悬挂传单
- 活动海报微微旋转 1-2° 模拟被钉在板上的自然角度
- 底部便签纸条（淡黄色 `bg-yellow-50`）：📅 日期 🕐 时间 📍 地点 💰 价格 👥 人数
- 6s 切换淡入淡出

**Card：票根/传单卡片**
- 顶部虚线撕边（`border-dashed border-t`）
- 活动海报 + 类型/标题/日期/地点
- hover 微上浮 + 虚线变实线 + 轻微旋转（像被风吹动的传单）

## 共通设计模式

- **Hero**：每品类独立 Hero 组件，自动播放轮播/展示，数据源 `getFeatured({type})` → fallback `getHotItems()`
- **Card**：每品类独立 Card 组件，hover 交互动画通过 `transitions.css` 统一管理
- **Editor**：每品类独立编辑字段组件，在 `AdminItemForm.vue` 中动态路由
- **types.js**：所有品类元数据集中管理（emoji/accent/heroColor/light/pagBg）
