## 三、Hero — "本月精选"展示台

**不是轮播**，是书店入口那张平铺展示台——3-4 本书封面朝上平放，带店员手写推荐卡：

```
┌─────────────────────────────────────────────────┐
│  深木色背景 + 柔光 radial-gradient 聚光           │
│                                                 │
│  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐       │
│  │ 📖   │  │      │  │      │  │      │       │
│  │ 三体  │  │人类简史│  │ EVA  │  │ 沙丘 │       │
│  │刘慈欣 │  │赫拉利 │  │贞本  │  │赫伯特│       │
│  │科幻   │  │历史   │  │漫画  │  │科幻  │       │
│  └──────┘  └──────┘  └──────┘  └──────┘       │
│  ┌─ 推荐卡 ─┐                                   │
│  │ 本月精选  │   ← 歪斜手写纸条 📝                │
│  └──────────┘                                    │
└─────────────────────────────────────────────────┘
```

**设计细节**：

| 元素 | 实现 |
|------|------|
| 背景 | 深木色渐变 + `radial-gradient(ellipse at 50% 30%, rgba(251,191,36,0.08) 0%, transparent 70%)` 柔光 |
| 书封面 | `aspect-[3/4]` 带 3D 书脊（`border-r-[3px] border-r-black/20` + 顶部高光线） |
| 推荐卡 | 奶油色纸条，`transform: rotate(-1deg)`, `font-serif italic`，模拟店员手写 |
| 数据源 | `getFeatured({type:'book'})` → fallback `getHotItems()` |
| 空状态 | 静默隐藏（`v-else />`） |

## 四、Card — 书架展示卡

每本书像插在书架上一样，露封面 + 书脊，hover 微微抽出：

```
┌──────────────────────────────────────────────────┐
│ ┌────────┐                                       │
│ │        │  📚 书籍  ·  科幻                      │
│ │  封面  │  三体                                  │
│ │ 3D侧边 │  刘慈欣  ·  312页  ·  2008             │
│ │        │  文化大革命中人类向宇宙发出第一声啼鸣...  │
│ └────────┘                        试读预览 →     │
│ ═══════════════════ 书架隔板 ═══════════════════  │
└──────────────────────────────────────────────────┘
```

| 区域 | 内容 | 数据来源 |
|------|------|---------|
| 左侧 1/4 | 书封面 + 3D 书脊侧边 + 顶部高光线 | `wideCoverUrl \|\| coverUrl`，`aspect-[3/4]` |
| 右上 | 品类标签「书籍」+ 分类（科幻/文学/历史） | `info_json.category` |
| 第二行 | 书名（大号 serif 字体） | `title` |
| 第三行 | 作者 · 页码 · 年份（暖棕色徽章） | `info_json.author`/`pages`/`year` |
| 描述 | 1-2 行简介 | `description` |
| 底部 | 「试读预览 →」 | 静态装饰（后续接试读功能） |
| 底部线 | 深木色书架隔板条 | 纯 CSS 装饰 |

**Hover 效果**：
- 封面微放大 `scale-105`
- 整卡略微右移 `translateX(4px)`，模拟抽书
- 阴影加深 `hover:shadow-lg`
- 书架隔板不变

**布局**：`space-y-4` 单列垂直，每本一本

## 七、特点

- **首个浅色品类**：`light:true` 在 types.js 已定义，List.vue 自动适配白底+黑字，无需额外判断

- **抽书动画**：hover 时 `translateX(4px)`，书架隔板保持不动
