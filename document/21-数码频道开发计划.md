## 二、视觉设计：产品聚焦展台

### 设计理念

借鉴 Apple/Dyson 官网产品页——大尺寸产品渲染图居中 + 极简排版 + 淡入轮播。

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  ┌──────────────────────────────────────────┐   │
│  │                                          │   │
│  │          PRODUCT IMAGE                   │   │
│  │       （全幅产品渲染图）                    │   │
│  │                                          │   │
│  └──────────────────────────────────────────┘   │
│                                                 │
│     ◄    ● ● ● ● ● ● ● ● ●    ►              │
│          Sony WH-1000XM5                        │
│          主动降噪 · 蓝牙5.2 · 30h续航            │
└─────────────────────────────────────────────────┘
```

### Hero 规格

| 属性 | 说明 |
|------|------|
| 背景 | 浅灰白 `from-gray-50 via-white to-gray-100`，`light:true` 匹配品类调性 |
| 图片 | `wideCoverUrl` 优先 > `coverUrl` fallback，`max-w-3xl` 居中 |
| 轮播 | `Vue Transition` 交叉淡入淡出（`mode="out-in"`），5s 自动播放 |
| 信息条 | 底部半透明白色毛玻璃（`backdrop-blur` + `bg-white/70`）显示品名/品牌 + 特性标签 |
| 指示器 | cyan-500 高亮 dot，可点击跳转 |
| 数据源 | `getFeatured({type:'digital'})` → fallback `getHotItems({type:'digital',limit:8})` |
| hover | 产品图 `hover:scale-[1.02]` 微放大，`transition duration-700` |

### 可选增强

- 产品图 `mix-blend-mode: multiply` 融入浅色背景
- 顶角添加细线边框（`border border-gray-200`）增加画框感

## 三、DigitalCard — 参数卡片（Spec Card）

### 设计理念

科技媒体评测卡片风格——左侧缩略图 + 右侧参数信息。干净、信息密度适中。

```
┌──────────────────────────────────────────┐
│ ┌────────┐                               │
│ │        │  AirPods Pro 2                │
│ │  📱    │  Apple · 2022                 │
│ │        │                               │
│ └────────┘  [空间音频] [自适应降噪]        │
│             [MagSafe] [蓝牙5.3]           │
└──────────────────────────────────────────┘
```

### 卡片规格

| 属性 | 说明 |
|------|------|
| 布局 | 横向 `flex`，左图 `w-28 h-28` 正方形，右区 `flex-1` |
| 背景 | `bg-white` + `border border-gray-100`，hover 时 `border-cyan-400` + `shadow-lg shadow-cyan-100` |
| 图片 | `coverUrl`，`object-cover`，`rounded-lg` |
| 品牌 | `text-xs text-gray-400 uppercase tracking-wide` |
| 标题 | `text-lg font-semibold text-gray-900` |
| 年份 | `text-xs text-gray-400`，与品牌同行 |
| 特性 | 小标签 `bg-cyan-50 text-cyan-700 text-xs px-2 py-0.5 rounded`，最多显示 4 个 |
| hover | 边框高亮 `border-cyan-400` + `shadow-lg` + 整体上浮 `-translate-y-1` |

## 四、DigitalEditorFields — 管理员编辑器

### 字段设计

| 字段 | 类型 | 说明 |
|------|------|------|
| 品牌 (brand) | `el-input` | 产品品牌，如 Apple、Sony |
| 类别 (category) | `el-select` | 预置选项：耳机/键盘/鼠标/显示器/手机/平板/笔记本/音箱/相机/其他 |
| 年份 (year) | `el-input-number` | 发布年份 |
| 特性 (features) | `el-input` | 逗号分隔的特性标签，如"主动降噪,蓝牙5.2,30h续航" |
| B站视频 | `el-input` | B站评测/开箱视频 URL |
| YouTube | `el-input` | YT 评测视频 URL |

## 五、数据采集方案

数码品类**没有**类似 Steam/iTunes/TMDB 的免费开放 API。方案如下：

### 当前方案：手动录入（推荐）

| 特性 | 说明 |
|------|------|
| 模式 | 参考模型频道——管理员后台手动添加 |
| 数据量 | 初期 10 款 + 后续按需扩充 |
| 信息填写 | 品牌 / 类别 / 年份 / 特性 + 上传产品图 |
| 优势 | 数据精准可控，不依赖外部 API 稳定性 |

### 备选方案：数据抓取（后期可选）

| 源 | 可行性 | 说明 |
|------|--------|------|
| 什么值得买 (SMZDM) | ⭐⭐ | 产品库有参数/价格/图片，需解析 HTML，无公开 API |
| 中关村在线 (ZOL) | ⭐⭐ | 参数最全的产品库，同样需爬虫解析 |
| GSMArena | ⭐⭐ | 手机参数全，有非官方 API，仅限手机品类 |
| 京东/天猫 | ⭐ | 反爬严格，不稳定 |

**结论**：初期走手动录入模式。后续如果需要自动化，可基于 SMZDM 或 ZOL 产品页编写简易爬虫，但优先级低。
