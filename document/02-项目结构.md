# 项目结构

> 完整目录树，与代码实际一致。

---

## 根目录

```
DemoNet/
├── .env.example              # 环境变量模板
├── .gitignore
├── docker-compose.yml        # 开发环境（MySQL + Redis + RabbitMQ）
├── docker-compose.prod.yml   # 生产环境（5-service 全栈）
├── SETUP.md                  # 新电脑环境搭建指南
├── backend/                  # Spring Boot 后端
├── frontend/                 # Vue 3 前端
├── document/                 # 项目文档
└── scripts/                  # 环境初始化 + 一键启动脚本
```

---

## 后端 `backend/src/main/java/com/example/demonet/`

```
DemoNetApplication.java          ← 入口

config/
├── SecurityConfig.java          ← Spring Security + JWT 无状态配置
├── MybatisPlusConfig.java       ← MyBatis-Plus 分页插件
├── RabbitMQConfig.java          ← RabbitMQ 队列/交换机定义
├── RedisConfig.java             ← Redis 序列化 + 缓存错误降级
└── WebConfig.java               ← CORS 跨域配置

security/
├── JwtTokenProvider.java        ← JWT 生成/验证
└── JwtAuthFilter.java           ← JWT 请求过滤器

entity/                          ← 数据库实体（5 个）
├── Item.java
├── User.java
├── UserItem.java
├── Tag.java
└── Review.java

mapper/                          ← MyBatis-Plus Mapper（5 个）
├── ItemMapper.java
├── UserMapper.java
├── UserItemMapper.java
├── TagMapper.java
└── ReviewMapper.java

service/                         ← 业务逻辑（15 个）
├── ItemService.java             ← 核心 CRUD + 搜索/推荐/热门
├── AuthService.java             ← 注册/登录 JWT
├── AdminService.java            ← 管理后台（审核/轮播/抓取触发/CRUD）
├── DataImportService.java       ← 种子数据导入（~51KB，116 条种子）
├── SteamService.java            ← Steam Store API
├── TMDBService.java             ← TMDB API v3
├── IGDBService.java             ← IGDB API v4（Twitch 游戏数据库）
├── AniListService.java          ← AniList GraphQL（动漫数据）
├── BangumiService.java          ← Bangumi 中文动漫数据
├── ItunesService.java           ← iTunes Search API（音乐数据）
├── FetchConsumer.java           ← RabbitMQ 消费者（异步抓取）
├── ScheduledFetchService.java   ← 定时抓取任务
├── TagService.java
├── UserItemService.java
└── ReviewService.java

controller/                      ← REST API（7 个）
├── ItemController.java          ← 公开 API（列表/详情/搜索/热门/推荐/精选）
├── AuthController.java          ← 注册/登录/当前用户
├── AdminController.java         ← 管理后台（CRUD/审核/抓取/轮播/标签/上传/统计）
├── TagController.java           ← 公开标签 API
├── UserItemController.java      ← 收藏 API
├── ReviewController.java        ← 评论 API
└── GlobalExceptionHandler.java  ← 统一异常处理

dto/
└── UserDTO.java

resources/
├── application.yml              ← 主配置（环境变量占位）
├── application-example.yml      ← 配置示例
├── schema.sql                   ← DDL 建表脚本（6 表）
└── data/seed.sql                ← SQL 种子数据
```

## 前端 `frontend/src/`

```
main.js                           ← 应用入口
App.vue                           ← 根组件
style.css                         ← 全局样式

router/index.js                   ← Vue Router（含 Admin 守卫）

stores/auth.js                    ← Pinia 认证状态

constants/types.js                ← 10 品类元数据（emoji/accent/heroColor/light）

styles/transitions.css            ← 品类专属动画

api/                              ← Axios 封装（6 个）
├── request.js                    ← Axios 实例 + JWT 拦截器
├── item.js                       ← 内容 API
├── auth.js                       ← 认证 API
├── tag.js                        ← 标签 API
├── admin.js                      ← 管理后台 API
└── review.js                     ← 评论 API

components/                       ← 组件
├── AppNavbar.vue                 ← 导航栏
├── AppFooter.vue                 ← 页脚
├── AppCard.vue                   ← 通用卡片（fallback）
├── TypeIcon.vue                  ← Element Plus 品类图标动态组件
├── SearchBar.vue                 ← 搜索栏
├── ReviewSection.vue             ← 评论组件
├── HelloWorld.vue                ← Vue 模板示例
│
├── category/                     ← 品类专属 Hero + Card（21 个）
│   ├── GameHero.vue              ← 游戏：无限循环轮播（暗绿+霓虹）
│   ├── GameCard.vue              ← 游戏：竖版卡片 + DEMO 徽章
│   ├── MovieHero.vue             ← 电影：全屏 Banner 轮播
│   ├── MovieCard.vue             ← 电影：横向宽卡
│   ├── MoviePosterCard.vue       ← 电影：3D 翻转海报卡
│   ├── AnimeHero.vue             ← 动漫：紫/粉调轮播
│   ├── AnimeCard.vue             ← 动漫：灰度 hover 卡
│   ├── BoardgameHero.vue         ← 桌游：游戏店橱窗（琥珀暖色）
│   ├── BoardgameCard.vue         ← 桌游：盒装展示卡 + 规则视频
│   ├── ModelHero.vue             ← 模型：格纳库陈列架
│   ├── ModelCard.vue             ← 模型：全宽画廊卡
│   ├── BookHero.vue              ← 书籍：书店展示台 + 3D 书脊
│   ├── BookCard.vue              ← 书籍：书架卡（抽书 hover）
│   ├── MusicHero.vue             ← 音乐：CoverFlow 3D 封面墙
│   ├── MusicCard.vue             ← 音乐：正方形专辑卡
│   ├── DigitalHero.vue           ← 数码：产品聚焦展台（暗青渐变）
│   ├── DigitalCard.vue           ← 数码：参数横卡
│   ├── CoffeeHero.vue            ← 咖啡：黑板手写菜单
│   ├── CoffeeCard.vue            ← 咖啡：咖啡豆标签卡（牛皮纸）
│   ├── OfflineHero.vue           ← 线下：软木公告板（图钉+传单）
│   └── OfflineCard.vue           ← 线下：票根传单卡
│
└── admin/                        ← 管理后台组件
    ├── AdminDashboard.vue
    ├── AdminItemList.vue         ← 内容列表（卡片/表格双模式）
    ├── AdminItemForm.vue         ← 内容编辑器（含品类专属编辑器路由）
    ├── AdminFetchPanel.vue       ← 数据抓取面板（6 数据源）
    ├── AdminCarouselManager.vue  ← 轮播管理
    ├── AdminTagManager.vue       ← 标签管理
    │
    └── 品类专属编辑器字段（11 个）
        ├── GameEditorFields.vue
        ├── MovieEditorFields.vue
        ├── AnimeEditorFields.vue
        ├── BoardgameEditorFields.vue
        ├── ModelEditorFields.vue
        ├── BookEditorFields.vue
        ├── MusicEditorFields.vue
        ├── DigitalEditorFields.vue
        ├── CoffeeEditorFields.vue
        ├── OfflineEditorFields.vue
        └── GenericEditorFields.vue  ← 兜底编辑器

views/                            ← 页面视图（8 个）
├── Home.vue                      ← 首页（品类入口 + 精选推荐）
├── List.vue                      ← 品类列表页（动态路由 type）
├── Detail.vue                    ← 详情页（~46KB）
├── Search.vue                    ← 搜索页
├── Login.vue                     ← 登录
├── Register.vue                  ← 注册
├── Profile.vue                   ← 个人主页
└── Admin.vue                     ← 管理后台主页
```
