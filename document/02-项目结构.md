# 项目结构

> 完整目录树，与代码实际一致（截至 2026-07-04）。

---

## 根目录

```
DemoNet/
├── .env.example              # 环境变量模板（含全部变量说明）
├── .env                      # 本地实际值（gitignored）
├── .gitignore
├── docker-compose.yml        # 开发环境（MySQL + Redis + RabbitMQ）
├── docker-compose.prod.yml   # 生产环境（5-service 全栈）
├── SETUP.md                  # 新电脑环境搭建指南
├── AGENTS.md                 # AI 协作规范与技术栈速览
├── demonet.sql               # 数据库完整备份（含种子数据）
├── backend/                  # Spring Boot 后端
├── frontend/                 # Vue 3 前端
├── document/                 # 项目文档
└── scripts/                  # 环境初始化 + 一键启动脚本
    ├── setup-env.sh          # 生成 .env 文件
    └── start-dev.sh          # 一键启动全栈
```

## 后端 `backend/src/main/java/com/example/demonet/`

```
DemoNetApplication.java          ← 入口（含 @PostConstruct 清缓存）

config/
├── SecurityConfig.java          ← Spring Security + JWT 无状态配置
├── MybatisPlusConfig.java       ← MyBatis-Plus 分页插件
├── RabbitMQConfig.java          ← RabbitMQ 队列/交换机定义（7条队列）
├── RedisConfig.java             ← Redis 序列化 + 缓存错误降级
└── WebConfig.java               ← CORS 跨域配置

security/
├── JwtTokenProvider.java        ← JWT 生成/验证（HMAC-SHA256）
└── JwtAuthFilter.java           ← JWT 请求过滤器

entity/                          ← 数据库实体（5 个 MyBatis-Plus 实体）
├── Item.java                    ← 核心内容（含 recommendations/hot_boost 字段）
├── User.java                    ← 用户（含 enabled 字段）
├── UserItem.java                ← 收藏关联
├── Tag.java
└── Review.java

mapper/                          ← MyBatis-Plus Mapper（5 个）
├── ItemMapper.java
├── UserMapper.java
├── UserItemMapper.java
├── TagMapper.java
└── ReviewMapper.java

dto/
├── UserDTO.java                 ← 用户响应对象（屏蔽密码等敏感字段）
├── LoginRequest.java            ← 登录请求（@NotBlank @Size 校验）
├── RegisterRequest.java         ← 注册请求（@NotBlank @Email @Size 校验）
└── ReviewCreateRequest.java     ← 评论创建请求（@NotBlank 校验）

service/                         ← 业务逻辑（16 个）
├── ItemService.java             ← 核心 CRUD + 搜索/推荐/热门（含热度公式）
├── AuthService.java             ← 注册/登录/JWT + 保留用户名校验
├── AdminService.java            ← 管理后台（审核/轮播/CRUD/统计/批量操作）
├── DataImportService.java       ← 种子数据导入（CommandLineRunner，~58KB，130+ 条）
├── SteamService.java            ← Steam Store API（含 SteamGridDB 封面回退）
├── SteamGridDBService.java      ← SteamGridDB API（竖版封面获取）
├── TMDBService.java             ← TMDB API v3（电影/剧集）
├── IGDBService.java             ← IGDB API v4（游戏，Twitch OAuth）
├── AniListService.java          ← AniList GraphQL（动漫）
├── BangumiService.java          ← Bangumi 中文动漫数据
├── ItunesService.java           ← iTunes Search API（音乐）
├── FetchConsumer.java           ← RabbitMQ 消费者（异步抓取 + 自动打标签）
├── ScheduledFetchService.java   ← 定时抓取任务（框架就绪，待接入具体逻辑）
├── TagService.java              ← 标签 CRUD + ensureTag/addItemTag
├── UserItemService.java         ← 收藏 API
└── ReviewService.java           ← 评论 API

controller/                      ← REST API（7 个 + 1 异常处理器）
├── ItemController.java          ← 公开 API（列表/详情/搜索/热门/推荐/精选）
├── AuthController.java          ← 注册（IP限流+Turnstile+邀请码）/登录/当前用户/公开配置
├── CategoryController.java      ← 公开品类可见性 API
├── AdminController.java         ← 管理后台（全部 Admin 操作，24KB）
├── TagController.java           ← 公开标签 API
├── UserItemController.java      ← 收藏 API
├── ReviewController.java        ← 评论 API
└── GlobalExceptionHandler.java  ← 统一异常处理（Duplicate/AccessDenied/Validation/Runtime/Generic）

resources/
├── application.yml              ← 主配置（全部敏感值从环境变量注入）
├── application-example.yml      ← 配置示例（参考用）
├── schema.sql                   ← DDL 建表脚本（含 items/users/user_items/tags/item_tag_mapping/reviews/category_settings）
└── data/seed.sql                ← SQL 种子数据（部分品类基础数据）
```

> **注意**：`invite_codes` 和 `app_settings` 表目前由 `DataImportService` 在启动时动态建立，暂未写入 `schema.sql`。

## 前端 `frontend/src/`

