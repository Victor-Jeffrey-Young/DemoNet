## 一、定位分析

### 不是"大众点评"

游戏频道不做"评分排名"，电影频道不做"影评打分"。线下频道同理——**不做评价，做预览**。

| 品类 | 「试玩」形式 |
|------|-------------|
| 游戏 | 预告片 + Steam 试玩 Demo |
| 电影 | 预告片 + 剧情简介 |
| 书籍 | 前 20 页试读 + EPUB 阅读器 |
| 线下 | **剧情试读 + 活动预览 + 现场实拍** |

核心理念：用户在决定"花时间/花钱出门"之前，能先感受一下这个地方/活动是什么样。

### 内容类型

| 类型 | 预览内容 | 示例 |
|------|----------|------|
| 密室逃脱 | 剧情背景 + 难度/恐怖度 + 人数 | 《弥生》日式惊悚，4-6人，恐怖★★★☆ |
| 沉浸式剧场 | 剧情概要 + 卡司/场地 | 《不眠之夜》沉浸式戏剧，Mckittrick酒店 |
| 展览 | 预览照片 + 主题 + 展期 | teamLab、敦煌光影展、安藤忠雄回顾 |
| 快闪市集 | 主题 + 摊主阵容 + 时间 | 凡几市集、银盐复古市集、伍德吃托克 |
| Live/音乐现场 | 歌单试听 + 场地氛围 | 万青巡演、爵士酒吧演出、音乐节 |
| 工作坊 | 做什么 + 学什么 + 带走什么 | 陶艺拉坯、手冲咖啡课、暗房胶片冲扫 |
| 城市探索 | 路线地图 + 沿途亮点 | Citywalk 武康路建筑漫游、苏州河夜跑 |

## 二、视觉设计：社区活动公告板

### 理念

模拟咖啡馆/社区空间墙上的**软木公告板（Cork Board）**——钉着活动传单、海报、票根。温暖的、有发现感的、像周末出门前翻看一样的体验。

```
┌─────────────────────────────────────────────────┐
│  ┌──────────────────────────────────────────┐   │
│  │  📌 本周精选                               │   │
│  │                                          │   │
│  │  ┌────────────┐  ┌────────────┐         │   │
│  │  │  📷        │  │  📷        │         │   │  ← 活动海报/传单
│  │  │  活动海报   │  │  活动海报   │  ...    │   │     钉在软木板上
│  │  │            │  │            │         │   │
│  │  └────────────┘  └────────────┘         │   │
│  │                                          │   │
│  │    📅 2024.12.25  🕐 19:00-21:00        │   │
│  │    📍 上海西岸美术馆  B1 展厅             │   │
│  │    💰 ¥128/人  👥 4-6人                │   │
│  └──────────────────────────────────────────┘   │
│         ◄    ● ● ●    ►                        │
└─────────────────────────────────────────────────┘
```

## 三、OfflineHero — 软木公告板

### 规格

| 属性 | 说明 |
|------|------|
| 背景 | 软木纹理底 `bg-[#c4a47a]`，CSS `repeating-linear-gradient` 模拟木纹/软木颗粒 |
| 装饰 | 红色图钉📌在四角和传单边缘；麻绳 + 小木夹悬挂传单 |
| 标题 | "📌 本周精选"，手写体或打字机字体（`font-mono`），白色粉笔/便签纸风格 |
| 图片展示 | 活动海报/现场照片，`max-w-md` 居中，微微旋转 1-2° 模拟被钉在板上的自然角度 |
| 信息条 | 底部便签纸条（淡黄色 `bg-yellow-50`），手写信息：📅日期 🕐时间 📍地点 💰价格 👥人数 |
| 轮播 | `Vue Transition` 交叉淡入淡出，6s 切换，模拟翻看公告板 |
| 指示器 | 图钉样式圆点，active 为 indigo 高亮 |
| 数据源 | `getFeatured({type:'offline'})` → fallback `getHotItems({type:'offline',limit:6})` |

## 四、OfflineCard — 票根/传单卡片

### 理念

每张卡片是一张**活动票根或撕下来的传单**——虚线撕边、日期时间醒目、能感受到"这是一张要去现场的东西"。

```
┌────────────────────────────┐
│ ┌──────────────────────┐   │
│ │                      │   │
│ │    活动海报/照片       │   │
│ │                      │   │
│ └──────────────────────┘   │
│                            │
│  🏛️ 沉浸式剧场              │  ← 类型标签
│  《不眠之夜》                │  ← 标题
│                            │
│  📅 2024.12.20 - 2025.3   │
│  📍 上海·麦金侬酒店         │
│                            │
│  [沉浸式] [戏剧] [SleepNoMore]│  ← 标签
├────────────────────────────┤
│      查看详情 →            │  ← 虚线撕边上方的提示
└────────────────────────────┘
```

### 规格

| 属性 | 说明 |
|------|------|
| 布局 | 纵向卡片 `flex flex-col` |
| 背景 | `bg-gray-900`，顶部虚线分离线（`border-dashed border-t border-gray-700` 模拟撕边） |
| 图片 | `aspect-[4/3]`，圆角 `rounded-t-lg`，`object-cover` |
| 类型标签 | `bg-indigo-500/10 text-indigo-300` 小标签 |
| 标题 | `text-lg font-bold text-white` |
| 日期/地点 | `text-xs text-gray-400`，前缀 emoji（📅📍💰） |
| 标签 | `text-[10px] bg-gray-800 text-gray-400 rounded` 小标签，最多 3 个 |
| hover | 整卡微上浮 + indigo 虚线变实线 + 轻微旋转 0.5°（像被风吹动的传单） |

## 五、OfflineEditorFields — 管理员编辑器

### 字段设计

| 字段 | 类型 | 说明 |
|------|------|------|
| 类型 (event_type) | `el-select` | 密室逃脱/沉浸式剧场/展览/市集/Live/工作坊/Citywalk/其他 |
| 地点 (venue) | `el-input` | 如：上海西岸美术馆 B1 展厅 |
| 日期 (date) | `el-input` | 如：2024.12.20 - 2025.03.15 |
| 时间 (time) | `el-input` | 如：10:00-18:00（周一闭馆） |
| 价格 (price) | `el-input` | 如：¥128/人 |
| 人数 (capacity) | `el-input` | 如：4-6人/场 |
| 难度/恐怖度 (difficulty) | `el-input` | 如：★★★☆（仅密室适用） |
| 体验亮点 (highlights) | `el-input` | 逗号分隔，如：互动装置,沉浸式,适合拍照 |
| B站视频 | `el-input` | B站 vlog/探展视频 |
| YouTube | `el-input` | YT 视频 |

### infoJson 结构

```json
{
  "event_type": "展览",
  "venue": "上海西岸美术馆",
  "date": "2024.12.20 - 2025.03.15",
  "time": "10:00-18:00（周一闭馆）",
  "price": "¥128/人",
  "capacity": "",
  "difficulty": "",
  "highlights": "互动装置,沉浸式,适合拍照,亲子友好",
  "videos": {
    "bilibili": "",
    "youtube": ""
  }
}
```

## 七、数据采集

线下体验同样**无公开 API**，走手动录入。

| 源 | 可行性 | 说明 |
|------|--------|------|
| 大麦/猫眼 | ⭐ | 无公开 API，反爬严格 |
| 小红书/点评 | ⭐ | 反爬严格 |
| 各场馆/活动官网 | ⭐⭐ | 信息分散，需逐个录入 |
| 手动录入 | ⭐⭐⭐ | 最实际，数据量不大 |
