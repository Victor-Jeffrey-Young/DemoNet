## 数据库表（9 张）

> schema 真相来源：`backend/src/main/resources/schema.sql` + 运行时 ALTER TABLE（用 DataImportService CommandLineRunner 补全）。

---

### items — 核心内容表

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | BIGINT PK AUTO_INCREMENT | 自增主键 |
| `type` | VARCHAR(20) NOT NULL | 品类：game / movie / anime / boardgame / model / book / music / digital / coffee / offline |
| `title` | VARCHAR(255) NOT NULL | 标题 |
| `slug` | VARCHAR(255) UNIQUE | URL 友好标识（唯一） |
| `cover_url` | VARCHAR(512) | 横版封面图 |
| `wide_cover_url` | VARCHAR(512) | 宽幅封面（备选） |
| `poster_url` | VARCHAR(512) | 竖版海报图（SteamGridDB / 管理员上传） |
| `media_url` | VARCHAR(512) | 视频/媒体 URL |
| `external_link` | VARCHAR(512) | 外部商店链接 |
| `description` | TEXT | 描述 |
| `info_json` | JSON | 品类特定结构化数据（见下方） |
| `external_id` | VARCHAR(100) | Steam AppID / TMDB ID 等 |
| `source` | VARCHAR(30) DEFAULT 'manual' | 数据来源：manual / steam / tmdb / itch 等 |
| `status` | TINYINT DEFAULT 1 | 1=online（已发布）, 0=offline（待审核） |
| `carousel_order` | INT NULL | NULL=不在轮播中, 0,1,2...=轮播排序 |
| `recommendations` | INT DEFAULT 0 | Steam 好评数，用于热度排序 |
| `hot_boost` | INT DEFAULT 0 | 管理员手动热度加权（新游戏自动设 200） |
| `created_at` | DATETIME DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| `updated_at` | DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

**索引**：`idx_type(type)`, `idx_slug(slug)`

**热度排序公式**：
```sql
热度分 = (LN(recommendations + 1) * 100 + hot_boost + recent_reviews * 500)
         / GREATEST(DATEDIFF(NOW(), created_at), 1)
```

> **重要**：`status` 使用 TINYINT。`pending` 接口查 `status=0`，`approve` 设为 `1`。

### category_settings — 品类可见性与排序

| 字段 | 类型 | 说明 |
|------|------|------|
| `type` | VARCHAR(20) NOT NULL PRIMARY KEY | 品类标识（game/movie/anime...） |
| `visible` | TINYINT NOT NULL DEFAULT 1 | 1=前台显示, 0=隐藏 |
| `sort_order` | INT NOT NULL DEFAULT 0 | 首页和导航栏排列顺序（升序） |

**初始数据**（INSERT IGNORE）：game(10), movie(20), anime(30), boardgame(40), model(50), book(60), music(70), digital(80), coffee(90), offline(100)

**API**：
- 公开：`GET /api/categories/visible` — 返回 visible=1 的品类（按 sort_order 排序）
- 管理：`GET /PUT /api/admin/categories/settings`

---

### invite_codes — 邀请码

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | BIGINT PK AUTO_INCREMENT | |
| `code` | VARCHAR(20) NOT NULL UNIQUE | 8 位大写 UUID 截取 |
| `used_by` | BIGINT NULL FK → users.id | 使用者 ID，NULL=未使用 |
| `created_at` | DATETIME DEFAULT CURRENT_TIMESTAMP | |

**业务逻辑**：
- `INVITE_ONLY` 开关存储在 `app_settings`，默认 false
- 管理员每次生成 5 个（可指定 count），使用后 `used_by` 不可复用
- 注册时若 `INVITE_ONLY=true` 则强校验邀请码

---

### app_settings — 系统配置键值对

| 字段 | 类型 | 说明 |
|------|------|------|
| `setting_key` | VARCHAR(100) NOT NULL PRIMARY KEY | 配置键 |
| `setting_value` | TEXT | 配置值 |

**已知配置键**：

| Key | 说明 | 默认值 |
|-----|------|--------|
| `STEAMGRIDDB_API_KEY` | SteamGridDB API 密钥 | 空 |
| `STEAM_API_KEY` | Steam Web API Key | 空 |
| `TMDB_API_KEY` | TMDB API v3 密钥 | 空 |
| `IGDB_CLIENT_ID` | Twitch IGDB Client ID | 空 |
| `IGDB_CLIENT_SECRET` | Twitch IGDB Client Secret | 空 |
| `TURNSTILE_SITE_KEY` | Cloudflare Turnstile 前端 key | 空 |
| `TURNSTILE_SECRET_KEY` | Cloudflare Turnstile 后端验证 key | 空 |
| `INVITE_ONLY` | 邀请码开关（"true"/"false"） | "false" |

> **注**：`app_settings`、`invite_codes` 尚未写入 schema.sql，由 `DataImportService` CommandLineRunner 在启动时 `CREATE TABLE IF NOT EXISTS` 创建（待 #A4 Flyway 迁移修复）。

## info_json 结构

`info_json` 使用 MySQL JSON 类型，新增品类无需改表。以下为各品类代表性字段：

### 游戏 (type=game)

```json
{
  "developer": "Supergiant Games",
  "publisher": "Supergiant Games",
  "genre": "动作, Rogue-like",
  "platform": "PC, PS5, Xbox",
  "release_date": "2024-09-17",
  "price": "198",
  "free": false,
  "demo_available": true,
  "demo_url": "steam://install/2716740",
  "has_benchmark": false,
  "benchmark_url": "",
  "languages": "简体中文, 英语*, 日语*",
  "screenshots": ["https://...jpg", "https://...jpg"],
  "features": ["单人", "成就", "Steam云存档"],
  "dlc": ["DLC名称1"],
  "videos": {
    "steam": "https://video.akamai.steamstatic.com/store_trailers/...",
    "youtube": "https://www.youtube.com/embed/...",
    "bilibili": "//player.bilibili.com/player.html?bvid=BV1..."
  }
}
```

> 语言字段：带 `*` 的支持全配音，不带 `*` 的仅字幕/界面。
