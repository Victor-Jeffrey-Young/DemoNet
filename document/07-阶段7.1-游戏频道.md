# 阶段 7.1：游戏频道视觉体验

> **状态**：✅ 已完成
> **数据**：14 款游戏，13 款有 YT+B站双视频源，5 款有试玩链接，全 Steam 真实封面

---

## 组件实现

### GameHero — 无限循环轮播

| 特性 | 实现 |
|------|------|
| 循环机制 | 3 份全量拷贝（24 张 DOM），中间区操作，边缘跳转不可见 |
| 驱动方式 | `transform: translateX()` GPU 加速，`requestAnimationFrame` 控制过渡 |
| 自动播放 | 3.5s 间隔 |
| 导航 | dot 指示器可点击跳转，箭头按钮 |
| 背景 | 暗绿 `radial-gradient` + CSS 网格点阵纹理 |
| 封面比例 | `aspect-[21/10]` 匹配 Steam 头图比例 (2.14:1)，`bg-top` 顶部对齐 |

### GameCard — 竖版卡片

| 特性 | 实现 |
|------|------|
| 尺寸 | 380px 宽度竖版布局 |
| Hover 效果 | 内容滑入视差，`700ms cubic-bezier(0.19,1,0.22,1)` |
| Stagger | 子元素延迟 80ms / 160ms 逐次入场 |
| DEMO 徽章 | 彩虹渐变边框脉冲动画，仅 `demo_available=true` 显示 |
| 封面 | Steam 真实封面，`bg-top` 对齐 |

### 试玩业务

- `info_json.demo_url` 存储 Steam `steam://install/APPID` 或 itch.io 链接
- 卡片 hover 滑入后显示"🎮 免费试玩"按钮
- 详情页显眼位置重复展示试玩入口
- 5 款有试玩：Hades II / 黑神话：悟空 / Metaphor: ReFantazio / Celeste / Buckshot Roulette

### 视频嵌入

- 详情页卡片头部嵌入 iframe 播放器
- 多源支持：优先 Bilibili → YouTube
- 右上角 [B站] [YT] 切换按钮
- `activeVideoUrl` + `isEmbed` 响应式控制

### Steam 数据管道

- Python 脚本调用 Steam Store API `store.steampowered.com/api/appdetails`
- 拉取 13/14 款真实封面（CDN URL）
- 提取 developer / genre / platform
- 填充 `info_json` 和 `cover_url`
