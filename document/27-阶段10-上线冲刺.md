## 10.1 种子图片替换（2h）

### 现状

116 条种子数据中大部分 `cover_url` 用 `https://picsum.photos/seed/xxx/600/400` 随机图。部分已通过 TMDB/iTunes/Steam 等抓取到真实图。

### 策略

不是手工替换 116 张，而是 **优先补覆盖率**：

| 优先级 | 品类 | 方案 |
|--------|------|------|
| P0 | game(14) | Steam 抓取已带真实图，验证完整性 |
| P0 | movie(32) | TMDB 抓取自带 w500 真实海报 |
| P0 | music(23) | iTunes 抓取自带 600px 专辑封面 |
| P1 | anime(14) | AniList/Bangumi 可补海报 |
| P1 | book(5) | 手动找封面，量少可接受 |
| P1 | model(10) | 手动机体图片，量少可接受 |
| P2 | boardgame(12) | BGG 可爬取盒图 |
| P2 | digital(3) | 官网/电商找产品图 |
| P2 | coffee/offline(16) | 网图填入 |

### 执行步骤

```bash
# 1. 检查各品类图片覆盖率
docker exec demonet-mysql mysql -uroot -pchangeme demonet -e "
SELECT type,
  COUNT(*) total,
  SUM(CASE WHEN cover_url LIKE 'https://picsum.photos%' OR cover_url IS NULL THEN 1 ELSE 0 END) missing
FROM items WHERE status=1 GROUP BY type;"

# 2. 优先重跑有真实 API 的抓取
# 游戏：Steam 抓取会带真实封面，删除旧数据重抓
# 电影：TMDB 抓取自带封面
# 音乐：iTunes 抓取自带封面
# 动漫：AniList/Bangumi 抓取

# 3. 手动品类：管理员后台逐条上传
# 打开 http://localhost:5173/admin → 编辑内容 → 上传封面/宽封面

# 4. 验证全部替换
docker exec demonet-mysql mysql -uroot -pchangeme demonet -e "
SELECT type,COUNT(*) FROM items WHERE (cover_url LIKE 'picsum%' OR cover_url IS NULL) AND status=1 GROUP BY type;"
# 预期：空输出（0 行）= 全部替换完成
```

```nginx
server {
    listen 80;

    location /.well-known/acme-challenge/ {

        return 301 https://$host$request_uri;

server {
    listen 443 ssl;
    server_name demonet.example.com;

    ssl_certificate     /etc/letsencrypt/live/demonet.example.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/demonet.example.com/privkey.pem;

    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml;

    location /api/ {
        proxy_pass http://backend:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location /uploads/ {
        proxy_pass http://backend:8080;
    }

    location / {
        root /usr/share/nginx/html;
        index index.html;
        try_files $uri $uri/ /index.html;
    }
}
```

## 10.3 改 admin 密码（15min）

### 方案：DataImportService 首次启动检测

在 `seedUser()` 中增加逻辑：如果 `admin` 用户已存在，打印警告但用**不再**用硬编码密码。改为从环境变量读取：

```java
private void seedUser() {
    Long count = jdbcTemplate.queryForObject("SELECT COUNT(*) FROM users", Long.class);
    if (count != null && count > 0) {
        // 检查 admin 密码是否为默认值——提示修改
        log.info("Users already exist, skipping seed.");
        return;
    }

    String adminPass = System.getenv("ADMIN_PASSWORD");
    if (adminPass == null || adminPass.isBlank()) {
        adminPass = "changeme";  // 仅首次无环境变量时用默认
        log.warn("ADMIN_PASSWORD not set, using default 'changeme'. CHANGE IT for production!");
    }
    jdbcTemplate.update(
        "INSERT INTO users (username, email, password_hash, role) VALUES (?, ?, ?, ?)",
        "admin", "admin@demonet.local", passwordEncoder.encode(adminPass), "ADMIN");
    log.info("Admin user created. CHANGE default password if using 'changeme'.");
}
```

### .env 配置

```bash
# .env
ADMIN_PASSWORD=生成一个随机32位密码
```

### 验证

```bash
curl -s -X POST http://localhost:8080/api/auth/login \
  -H 'Content-Type: application/json' \
  -d '{"username":"admin","password":"changeme"}'
# 预期：401 或 400（旧密码失效）
```

#### AuthControllerTest.java

```java
@SpringBootTest(webEnvironment = WebEnvironment.RANDOM_PORT)
@AutoConfigureMockMvc
class AuthControllerTest {

    @Autowired private MockMvc mockMvc;

    @Test
    void registerAndLogin_shouldReturnToken() throws Exception {
        String username = "test_" + System.currentTimeMillis();
        String body = "{\"username\":\"" + username + "\",\"password\":\"pass123\"}";

        // 注册
        mockMvc.perform(post("/api/auth/register")
                .contentType(MediaType.APPLICATION_JSON).content(body))
                .andExpect(status().isOk())
                .andExpect(jsonPath("$.token").exists())
                .andExpect(jsonPath("$.user.role").value("USER"));

        // 登录
        mockMvc.perform(post("/api/auth/login")
                .contentType(MediaType.APPLICATION_JSON).content(body))
                .andExpect(status().isOk())
                .andExpect(jsonPath("$.token").exists());
    }

    @Test
    void loginWithWrongPassword_shouldFail() throws Exception {
        mockMvc.perform(post("/api/auth/login")
                .contentType(MediaType.APPLICATION_JSON)
                .content("{\"username\":\"admin\",\"password\":\"wrong\"}"))
                .andExpect(status().is4xxClientError());
    }

    @Test
    void me_withoutToken_shouldFail() throws Exception {
        mockMvc.perform(get("/api/auth/me"))
                .andExpect(status().is4xxClientError());
    }
}
```

### 备份脚本 `scripts/backup.sh`

```bash
#!/bin/bash
BACKUP_DIR="/opt/demonet/backups"
mkdir -p "$BACKUP_DIR"
FILE="$BACKUP_DIR/demonet_$(date +%Y%m%d_%H%M).sql.gz"

docker exec demonet-mysql mysqldump -uroot -p"${MYSQL_ROOT_PASSWORD}" demonet | gzip > "$FILE"

# 保留最近 7 天
find "$BACKUP_DIR" -name "*.sql.gz" -mtime +7 -delete

echo "Backup: $FILE ($(du -h "$FILE" | cut -f1))"
```
