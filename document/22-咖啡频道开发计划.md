## 三、CoffeeHero — 黑板手写菜单

### 概念

模拟咖啡馆墙上的黑板菜单（Chalkboard Menu），深色板底 + 白色/彩色粉笔文字 + 手写风标题 + 擦除痕迹装饰。

### 规格

| 属性 | 说明 |
|------|------|
| 背景 | 深绿/深灰黑板色 `bg-[#1a3a2a]` 或 `bg-[#2d2d1a]`，边缘木框 `border-8 border-amber-900` |
| 标题 | "☕ Today's Recommendation"（粉笔手写体，`font-serif italic` + `text-amber-200`） |
| 装饰 | 擦除粉笔灰痕迹（CSS gradient 半透明白色斜线），右下角粉笔🧑‍🏫装饰 |
| 图片 | 咖啡豆/咖啡袋图，带麻布袋纹理背景 `bg-[#d4c5a9]`，`mix-blend-mode` 融入黑板 |
| 信息 | 产地 · 烘焙度 + 风味 emoji 标签（🍊🌸🍯🍫🌰）|
| 轮播 | `Vue Transition` 淡入淡出，5s 自动播放 |
| 指示器 | 白色粉笔点，active 为橙色高亮 `bg-orange-400` |
| 数据源 | `getFeatured({type:'coffee'})` → fallback `getHotItems({type:'coffee',limit:6})` |

### 可选装饰

- 黑板顶部"今日推荐"用真实的粉笔手写字体（Google Fonts: `Caveat` / `Patrick Hand` / `Kalam`）
- 黑板边缘放一个挂着的粉笔🧑‍🏫 + 板擦装饰
- 淡黄色暖光从上往下打（`radial-gradient` 模拟聚光灯）

## 四、CoffeeCard — 咖啡豆标签卡

### 概念

模拟精品咖啡豆包装袋上的标签（Coffee Label）—— 粗麻布/牛皮纸底色 + 圆角标签 + 手写风味描述。

### 规格

| 属性 | 说明 |
|------|------|
| 布局 | 竖向卡片 `flex flex-col`，`aspect-[3/4]` 或正方形 |
| 背景 | 牛皮纸色 `bg-[#c4a97d]` 或麻布袋 `bg-[#d4c5a9]` + 纸纹理（CSS radial-gradient 模拟纤维） |
| 图片区 | 上部 `aspect-square`，咖啡豆/袋图 `object-cover` |
| 标签区 | 下部信息区，米黄色 `bg-[#f5e6c8]` 或 `bg-amber-50`，圆角标签纸效果 |
| 产地 | `text-xs text-amber-800 uppercase tracking-wider` 粗体 |
| 品名 | `text-base font-bold text-gray-900` |
| 烘焙度 | 小圆点指示器：○浅烘 ◑中烘 ●深烘 |
| 风味 | emoji 风味标签 `bg-amber-100 text-amber-800`，最多显示 3 个 |
| hover | 卡片微微上浮 + 标签纸"掀起一角"阴影效果（`rotate-1` + `shadow-2xl`） |
| 外框 | 木质相框边框 `border-4 border-amber-900/60 rounded-lg` |

## 五、CoffeeEditorFields — 管理员编辑器

### 字段设计

| 字段 | 类型 | 说明 |
|------|------|------|
| 产地 (origin) | `el-select` | 预置选项：埃塞俄比亚/哥伦比亚/危地马拉/肯尼亚/巴西/印尼/哥斯达黎加/巴拿马/中国云南/其他 |

## 六、数据采集方案

咖啡豆同样**没有公开 API**，走手动录入模式。

| 源 | 可行性 | 说明 |
|------|--------|------|
| 精品咖啡烘焙商官网 | ⭐ | 每家独立网站，无统一 API |
| 咖啡豆评测网站 | ⭐⭐ | CoffeeReview.com 有评分和风味描述，但需爬虫 |
| 淘宝/京东咖啡豆 | ⭐ | 反爬严格 |
| 手动录入 | ⭐⭐⭐ | 最实际，数据量不大 |

**结论**：手动录入为主，参考模型/数码频道模式。8 款精选豆覆盖主流产区和烘焙度即可。

### 💡 「试玩」体验：冲煮指南 + 品鉴视频

咖啡没法像游戏一样"试玩"，但可以像电影频道嵌入预告片一样，提供两种"体验"形式：

| 形式 | 类比 | 实现方式 |
|------|------|----------|
| **冲煮文章** | 游戏的试玩 Demo | `info_json` 新增 `brew_guide` 字段，存 Markdown/HTML 冲煮教程（水温/粉水比/时间/步骤），详情页用 `v-html` 渲染成参数卡片 |
| **品鉴视频** | 电影的预告片 | 复用已有 `videos.bilibili` / `videos.youtube` + hero 区 iframe 嵌入（与游戏/电影一致的体验） |

**详情页布局设想**：

```
┌─────────────────────────────────────────────┐
│  ┌───────────────────────────────────────┐  │
│  │   品鉴视频 / 冲煮教学 (iframe/视频)     │  │  ← hero 区优先视频
│  └───────────────────────────────────────┘  │
│                                             │
│  ☕ 咖啡    manual                           │
│  埃塞俄比亚 耶加雪菲                  [收藏] │
│  柑橘与茉莉的经典组合...                     │

│  ┌──────┬──────┬──────┬──────┐             │
│  │ 产地 │烘焙度│处理法│ 品种 │             │
│  └──────┴──────┴──────┴──────┘             │

│  ☕ 风味   [🍊柑橘] [🌸茉莉] [🍯蜂蜜]        │
```

**`brew_guide` 字段结构**：

```json

  }
}
```

**编辑器扩展**：在 `CoffeeEditorFields.vue` 中新增「冲煮建议」区域（水温/粉水比/时间/步骤 4 个字段）。
