## 注册安全与提示系统重构（2026-07-03 续）

### 注册防护三层机制
- **IP 限流**：`AuthController.register()` 使用 Redis `INCR + EXPIRE`，同 IP 1 分钟最多 3 次
- **Cloudflare Turnstile**：前端 Register.vue 动态加载 widget → 后端 `POST siteverify` 验证
  - 未配置 secret key 时自动跳过
  - 网络不通时降级放行（不阻塞注册），只记日志
- **邀请码机制**：`invite_codes` 表 + 管理后台开关 + 注册校验
  - `INVITE_ONLY` 默认 false，开关实时生效无需重启
  - 管理后台可生成邀请码（每次 5 个，8 位 UUID）
  - 邀请码使用后标记 `used_by`，不可复用

### 保留用户名
- `AuthService` 注册时禁止 10 个系统保留字段：
  `admin`、`root`、`system`、`administrator`、`moderator`、`owner`、`superuser`、`guest`、`anonymous`、`api`（大小写不敏感）
