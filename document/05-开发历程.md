## 咖啡频道增强（2026-06-24）

### CQI 数据导入
- 从 [Coffee Quality Institute](https://github.com/jldbc/coffee-quality-database) 下载 1300+ 条阿拉比卡品测数据
- 新建 `seedCQICoffeeIfMissing()` 方法，CSV 解析 + 字段映射
- 筛选 SCA 总分 ≥ 86 分的精品咖啡，导入 52 条
- info_json 扩展：aroma / flavor / aftertaste / acidity / body / balance / uniformity / clean_cup / sweetness / cupper_points / total_cup_points 共 11 个评分字段
- 注意：CSV 首个字段为空引号 `""`，所有列索引需 +1

### 前端实现
- **SCA 雷达图**：ECharts 九维度雷达图，深棕背景 + 琥珀色描边
  - ECharts 5 tree-shaking 问题：`import from "echarts"` 不含雷达图组件，需改用 `echarts/dist/echarts.esm.js`
  - v-if 内 ref 绑定延迟：nextTick 不够，改为 100ms 轮询重试（最多 20 次）
- **CoffeeCard 增强**：左上角 ⭐ SCA 总分徽章（仅 CQI 数据）
- **冲煮计算器**：输入粉量 → 自动计算注水量、水温、建议时间
- **flavor 字段命名冲突**：CQI 数值 `flavor` 被风味描述字符串覆盖，改为 `flavor_notes` 存描述

## 安全与架构优化（2026-07-02）
